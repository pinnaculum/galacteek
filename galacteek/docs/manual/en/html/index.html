You can easily access this manual by typing manual:/ in the address bar of the browser.

About¶

galacteek is an experimental browser for the IPFS peer-to-peer network, written in Python with the Qt GUI framework. The goal is to provide a simple interface for publishing and accessing content over IPFS, accessible to most users.
